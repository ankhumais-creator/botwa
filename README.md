# Efficient WhatsApp CS Bot

## Cara Menjalankan Bot
1.  Buka folder ini di Terminal / VS Code.
2.  Jalankan perintah:
    ```bash
    npm start
    ```
3.  Buka browser dan akses: [http://localhost:3000](http://localhost:3000)

## Fitur
- **Dashboard**: Monitoring chat real-time.
- **Settings**: Ganti API Key, Model AI, dan System Prompt (Info Toko).
- **Human Mode**: Klik chat untuk balas manual atau matikan AI sementara.
- **Persistence**: Semua pengaturan disimpan otomatis.

## Catatan
- Bot ini berjalan di komputer Anda. Jangan tutup terminal agar bot tetap aktif.
